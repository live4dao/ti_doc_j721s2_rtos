◆ enableDOF

uint32_t Dof_Config::enableDOF

Set to 0x1 to enable DOF engine.

◆ width

uint32_t Dof_Config::width

Width of frame (in pixel) to be processed by Optical Flow engine. 2048 pixel max and 32 pixels min.

◆ height

uint32_t Dof_Config::height

Height of frame (in pixel) to be processed by Optical Flow engine. 1024 pixel max and 16 pixels min.

◆ enableSof

uint32_t Dof_Config::enableSof

sparse optical flow enable bit. Set to 0x1 to enable sparse optical flow processing.

◆ currentCensusTransform

uint32_t Dof_Config::currentCensusTransform

Census transform type for current image. 0: 24 bits generated for census transform by using a 5x5 neighborhood around the pixel. 1: 32 bits generated for census transform by using a 7x7 neighborhood around the pixel where only 8 pixels are used from the periphery.

◆ referenceCensusTransform

uint32_t Dof_Config::referenceCensusTransform

Census transform type for reference image. 0: 24 bits generated for census transform by using a 5x5 neighborhood around the pixel. 1: 32 bits generated for census transform by using a 7x7 neighborhood around the pixel where only 8 pixels are used from the periphery.

◆ maxMVsInSof

uint32_t Dof_Config::maxMVsInSof

Maximum number of MV output per line in case of sparse optical flow processing.

◆ motionSmoothnessFactor

uint32_t Dof_Config::motionSmoothnessFactor

Motion smoothness factor.

◆ iirFilterAlpha

uint32_t Dof_Config::iirFilterAlpha

Coefficient for IIR filter used for smoothing horizontal flow vector gradient. The usage can be illustrated as: SmoothU(i,j)= [ U(i,j)*alpha + SmoothU(i-1,j)*beta + round ]>>8 where beta = 256-alpha round = 128
