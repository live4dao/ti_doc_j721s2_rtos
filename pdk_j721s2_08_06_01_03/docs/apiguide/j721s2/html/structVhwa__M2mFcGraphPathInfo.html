Vhwa_M2mFcEdgeInfo Vhwa_M2mFcGraphPathInfo::edgeInfo[VHWA_FC_MAX_EDGES]

List of edges connecting VHWA Flex-Connect. Driver parses these edges and enables/disables input/output path in the appropriate VPAC module. This edge tells which module is connected to which module enabling output in edge start module and input in edge end module.
