Performance Measurements

The performace of various peripherals, drivers are highlighted under following sections. For detailed documentation and test reports refer MCUSW Documentation($PSDKRA_INSTALL_PATH/mcusw/mcusw_release_notes.html::Documentation)

  1. MCAL Can Profile
  2. XIP
  3. MCAL CDD Ipc Profile
  4. SPI based IPC
  5. Early CAN response time: 39.6 ms, including PMIC ramp time, on J7200 GP device (without BIST) and 94.3 ms on J721E.

Note: Reasoning behind worse numbers on J721E compared to previous release is:

  1. Had to disable the OSPI PHY, otherwise PHY tuning would be required, and
  2. HSM rearchitecture caused to have larger sizes for SBL and for application, since the sciclient must have code to service its own calls now on core mcu1_0.
